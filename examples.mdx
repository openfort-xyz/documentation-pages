# Examples

A list of examples you can start interacting with:

- Sample RainbowKit with Wagmi and NextAuth.js application: [GitHub source](https://github.com/openfort-xyz/samples/tree/main/rainbow-ssv-nextjs) and [video demo](https://youtu.be/rh2E02PATlU).
- Sample NextAuth.js with email/social login application: [GitHub source](https://github.com/openfort-xyz/samples/tree/main/ssv-social-nextjs) and [video demo](https://youtu.be/xBzl8hk__P8).
- How to use Web3Auth as key management and authentication solution: [GitHub source](https://github.com/openfort-xyz/samples/tree/main/web3auth-wagmi-nextjs).
- Learn how to create a Gaming Ecosystem creating your own OAuth: [GitHub source](https://github.com/openfort-xyz/samples/tree/main/oauth-game-ecosystem) and [video demo](https://youtu.be/s8wMDpTTzJE).
- Use an external KMS to sign transaction intents: [GitHub source](https://github.com/openfort-xyz/samples/tree/main/external-kms-game) and [video demo](https://youtu.be/uHigZXdTECw).
- Use Firebase Auth with Firestore and Wagmi: [GitHub source](https://github.com/openfort-xyz/samples/tree/main/firebase-wagmi-nextjs).
- Unity SDK Demo: [Video](https://www.youtube.com/watch?v=EQoj7Dl616E).
- API Walkthrough: [Video](https://www.youtube.com/watch?v=QDEIuSu4l7U).

Implemented your own library or an example you'd like to share? Send a link to your code to support@openfort.xyz and we'll be happy to add it to the list!
